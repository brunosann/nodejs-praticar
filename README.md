# Praticando Nodejs

# CRUD de Receitas

Node + Typescript

### Pré-requisitos globais:

`Node v16.13.2`
`npm i -g nodemon typescript ts-node`
`mysql`

### Instalação

`npm install`

### Para rodar o projeto

- Crie um `.env` na raiz do projeto e copie as variáveis de ambiente do `.env.example`

`npm run start-dev`
